However, we've added a couple of extra features to keep gameplay fun.
  • The Dragon Sword has been disabled.
  • All subweapons have been removed, except for the windmill shuriken.
  • B now throws a windmill shuriken.
  • Up + B now throws a "boost" windmill shuriken, causing a damage boost when it is caught.
  • Multiple swag stars now allowed onscreen via II, III and IV powerups.
